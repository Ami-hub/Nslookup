# nslookup
Find DNS records for a domain name!

## INSTALLING DEPENDENCIES
[Npcap](https://npcap.com/#download "Npcap\n")
```bash
pip install scapy
```

## HOW TO START
```bash
python nslookup.py <domain name>
```
or
```bash
python nslookup.py -type=PTR <ip address>
```
